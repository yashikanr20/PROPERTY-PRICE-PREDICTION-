# PROPERTY-PRICE-PREDICTION
A machine learning project that predicts property prices using a real-world dataset, built to explore end-to-end regression modeling — from data cleaning and EDA to feature engineering and model evaluation.
<br>
## Overview
This project builds a machine learning model to predict property prices based on features such as location, area, number of bedrooms/rooms, and other property characteristics. The goal is to help buyers, sellers, and real estate platforms estimate fair market value using data-driven predictions.
This project focuses on predicting property prices in various districts of California. The project will specifically utilize simple linear regression and multiple linear regression to address this regression task, ensuring proper data handling and evaluation of the models.
<br>
## Problem Statement
The objective is to predict the median house value in **California districts** based on features such as income, the number of rooms, geographical location, and proximity to the ocean. Given the dataset, we will develop regression models, evaluate their performance, and determine which model provides the best balance between predictive accuracy and interpretability.
